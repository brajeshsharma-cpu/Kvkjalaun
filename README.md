# Krishi Vigyan Kendra, Jalaun — Official Website

Modern, complete, responsive website. Deploy in 5 minutes using GitHub + Vercel.

---

## ⚡ Deploy in 3 Steps

### Step 1 — Create GitHub Repository
1. Go to **github.com** → Sign in → Click **"New Repository"**
2. Name: `kvk-jalaun-website` → Set **Public** → Click **"Create Repository"**
3. Upload **all files from this folder** (drag and drop them all)
4. Click **"Commit changes"**

### Step 2 — Deploy on Vercel (Free, ~60 seconds)
1. Go to **vercel.com** → Sign in with your GitHub account
2. Click **"Add New Project"** → Select `kvk-jalaun-website`
3. Click **"Deploy"** — no configuration needed
4. ✅ Your site is live at `https://kvk-jalaun-website.vercel.app`

### Step 3 — Custom Domain (Optional)
- In Vercel → Project Settings → Domains → Add `jalaun.kvk4.in`
- Update your domain DNS records as shown by Vercel

## ✏️ To Update Content

- **Add news**: Edit `activities.html` — copy an activity card block and update text
- **Add photos**: Edit `gallery.html` — add `<img src="YOUR_IMAGE_URL">` inside a `.gi` div
- **Update faculty**: Edit `faculty.html` — replace placeholder names and photos
- **Add documents**: Put PDFs in the project root, link them in `recruitment.html`
